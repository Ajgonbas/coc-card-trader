Clear everything?

This will wipe all card data, page names, and rewards from both the page and local storage. There's no undo — export a JSON first if you want a backup.

Load initial config?

This will refresh set names, card rarities, and rewards from the default configuration. Your card status (Have/Missing) and copy counts are not affected.

⚠️ This only works when accessed from ajgonbas.github.io/coc-card-trader. If you downloaded the file locally, use Load Album instead and load initial_config.json manually.
