Clear everything?

This will wipe all card data, page names, and rewards from both the page and local storage. There's no undo — export a JSON first if you want a backup.

Load initial config?

This will clear all your current data and replace it with the default album configuration (set names, card rarities, rewards).

⚠️ This only works when accessed from ajgonbas.github.io/coc-card-trader. If you downloaded the file locally, use Import JSON instead and load initial_config.json manually.
